# 📚 Multiple PDF Chat AI

AI-powered chatbot that allows users to upload multiple PDF files and ask questions about them.

## 🚀 Features
- PDF text extraction
- Text chunking
- Vector search (FAISS)
- LLM-based question answering
- Streamlit UI

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- GEMINI 

# ▶️Installation

## Install the required Python packages:
```bash
pip install -r requirements.txt
```

## ▶️ Run the app
```bash
streamlit run app.py
```

## Set up your Google API key from https://makersuite.google.com/app/apikey by creating a .env file in the root directory of the project with the following contents:

GOOGLE_API_KEY =<your-api-key-here>
